# KnowUrBites

Know Ur Bites is a dedicated web application designed to empower users to make healthier and safer food  choices. It covers five key aspects of food habits: safety (adulteration), healthy eating (swaps), seasonality, sustainability (leftover use), and health risks (allergies).The primary goal is to provide accessible, clear, and actionable information using an intuitive user interface.

# index.html is the home page.
